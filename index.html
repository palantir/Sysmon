Sysmon by palantir

Palantir System Monitoring Framework

The Palantir System Monitoring Framework is a framework for monitoring the host platform for a JavaVM and publishing that data via JMX Beans. Additionally, it contains an implementation of platform monitoring for Linux-based systems.

Dependencies

This project depends on Google's Guava library for its great extensions to the Java core. Additionally, Apache's commons-io and commons-lang are used for their elegant wrappering of some common idioms. Log4J is used as its logging framework. Finally, Junit 3.8 for unit testing.
